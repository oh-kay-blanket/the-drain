# Start the development server

This command will:
1. Start the Vite development server with hot module reloading
2. Open the application in your browser
3. Watch for file changes and auto-reload

Start the development server:

```bash
npm run dev
```

The application will be available at the URL shown in the terminal output (typically http://localhost:5173).

Press `Ctrl+C` in the terminal to stop the development server when you're done.
